# 권수현 포트폴리오

Flutter Web으로 개발한 개인 포트폴리오 웹사이트를 GitHub Pages를 통해 배포한 프로젝트입니다.

## 📌 프로젝트 소개

개인 역량, 프로젝트 경험, 기술 스택 등을 소개하기 위해 제작한 포트폴리오 웹사이트입니다.

Flutter를 활용하여 반응형 웹 환경을 구현하였으며, GitHub Pages를 통해 누구나 접근 가능한 형태로 배포하였습니다.

## 🚀 주요 기능

* 자기소개 및 프로필 제공
* 프로젝트 경험 소개
* 기술 스택 정리
* 반응형 웹 지원
* GitHub Pages 배포

## 🛠️ 기술 스택

* Flutter
* Dart
* Git
* GitHub Pages

## 📂 프로젝트 구조

```text
portfolio1/
├── assets/
├── canvaskit/
├── icons/
├── index.html
├── flutter.js
├── flutter_bootstrap.js
├── flutter_service_worker.js
├── manifest.json
└── favicon.png
```

## 🌐 배포

GitHub Pages를 활용하여 정적 웹사이트 형태로 배포하였습니다.

## 🎯 프로젝트 목적

개인 프로젝트와 활동 내역을 한곳에 정리하여 채용 담당자, 면접관, 협업자가 쉽게 확인할 수 있는 온라인 포트폴리오를 제공하는 것을 목표로 합니다.
